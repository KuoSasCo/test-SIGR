Sobre el proyecto

¿Cómo funciona
el sistema?

SIGR combina una Raspberry Pi con los servicios de inteligencia artificial de Amazon Web Services para clasificar residuos sólidos automáticamente. El modelo fue entrenado con imágenes reales de residuos sobre fondo negro.
